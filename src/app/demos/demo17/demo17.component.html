Demo 17 - Les Guards

Les Guards sont des mécanismes qui permettent de contrôler la navigation dans une application. Les Guards sont des classes qui implement l'interface 'CanActivate', 'CanActivateChild', 'CanDesactivate' ou 'CanLoad'.

CanActivate

Est utilisé pour déterminer si un utilisateur a le droit d'accéder à une page en vérifiant les permissions de l'utilisateur ou en effectuant des opérations d'authentification.

CanActivateChild

Est similaire à CanActivate, mais il est utilisé pour contrôler l'accès à une route enfant (un composant inclus dans un autre composant).

CanLoad

Est utilisé pour empêcher le chargement d'un module à moins que certains conditions soit remplies.

CanDesactivate

Est utilisé pour confirmer si un utilisateur peut quitter une page avant de la quitter réellement.
