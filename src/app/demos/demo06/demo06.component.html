Demo 06 - Les Pipes

Les pipes (ou filtres) sont fonctions qui permettent de transformer les données affichées dans la vue. Les pipes sont utilisés pour formater, trier, filtrer ou limiter les données affichées.

Les pipes sont utilisés dans les expressions d'interpolation {{ "{{ expression | nomPipe }}" }} ou dans les directives de liaison de propriété ([prop] = "expression | nomPipe")

Angular fournit un certain nombre de pipes intégrés, tels que "date", "uppercase", "lowercase", "currency", "decimal", "percent", "async", "json", ... Il est également possible de créer des pipes personnalisés pour répondre aux besoins spécifiques de l'application.

Exemples :

String :

UpperCasePipe

LowerCasePipe

TitleCasePipe

Avant : {{ maVariable1 }}

UpperCase : {{ maVariable1 | uppercase }}

LowerCase : {{ maVariable1 | lowercase }}

TitleCase : {{ maVariable1 | titlecase }}

Number :

DecimalPipe

minIntegerDigits . minFractionDigits - maxFractionDigits

Avant : {{ maVariable2 }}

Après : {{ maVariable2 | number }}

Après : {{ maVariable2 | number : '1.6-6' }}

Après : {{ maVariable2 | number : '5.10-15' }}

Après : {{ maVariable2 | number : '5.10-15' : 'fr-FR' }}

Date :

DatePipe

Avant : {{ maVariable3 }}

Après : {{ maVariable3 | date }}

Après : {{ maVariable3 | date : 'short' }}

Après : {{ maVariable3 | date : 'medium' }}

Après : {{ maVariable3 | date : 'long' }}

Après : {{ maVariable3 | date : 'dd/MM/yyyy' }}

Percent :

UpperCasePipe

Avant : {{ 0.2345 }}

Après : {{ 0.2345 | percent }}

Après : {{ 0.0345 | percent : '2.3-5' }}

Après : {{ 0.0345 | percent : '2.3-5' : 'en-US' }}

Currency :

CurrencyPipe

Avant : {{ 12.34 }}

Après : {{ 12.34 | currency }}

Après : {{ 12.34 | currency : 'EUR' }}

Après : {{ 12.34 | currency : 'JPY' }}

Après : {{ 12.34 | currency : 'GBP' : 'symbol' }}

Après : {{ 12.34 | currency : 'GBP' : 'code' }}

Après : {{ 12.34 | currency : 'GBP' : 'symbol-narrow' }}

Après : {{ 12.34 | currency : 'JPY' : 'symbol-narrow' }}

Après : {{ 12.34 | currency : 'EUR' : 'symbol-narrow' }}

Après : {{ 12.34 | currency : 'EUR' : 'symbol-narrow' : '4.4-6' }}

Après : {{ 12.34 | currency : 'EUR' : 'symbol-narrow' : '4.4-6' : 'en-US' }}

JSON :

JsonPipe

Avant : {{ maVariable4 }}

Après : {{ maVariable4 | json }}
