Demo 07 - Les Custom Pipes

Les customs pipes (ou pipes personnalisés) sont des pipes créés par les développeurs pour répondre aux besoins spécifiques à leur application. Les cutoms pipes sont crées en utilisant la class PipeTransform fournie par Angular, qui contient une méthode transform permettant de transformer les données en entrée.

Commande pour créer un pipe : ng generate pipe nom_pipe ou ng g p nom_pipe
